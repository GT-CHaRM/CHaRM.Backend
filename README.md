# CHaRM.Backend

## Running the Server

-   Make sure to have .NET Core SDK 2.2 installed.
-   Clone the repository

    ```sh
    git clone --recurse-submodules -j8 git://github.com/GT-CHaRM/CHaRM.Backend.git
    ```

-   Install all the dependencies

    ```sh
    dotnet restore
    ```

-   Run the server

    ```sh
    dotnet run
    ```
